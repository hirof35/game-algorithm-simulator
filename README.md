# Game Algorithm Simulator Suite

TypeScript と HTML5 Canvas を使用した、ゲームアルゴリズムのリアルタイム・視覚化シミュレーターです。

## 🚀 収録アルゴリズム
1. **Boids (群れ制御)**: 分離・整列・結合の3ルールによる個体群のシミュレーション
2. **A* (経路探索)**: マンハッタン距離を用いたリアルタイム最短ルート探索
3. **穴掘り法 (迷路生成)**: スタック（バックトラック）を活用した単一接続迷路の自動生成
4. **空間分割法 (衝突判定)**: 2Dグリッドハッシュを用いた $O(N)$ 前後への当たり判定高速化（2000粒子対応）

## 🛠️ 設計の特徴 (Architectural Design)
- **Interface Driven**: 全てのシミュレーターは `ISimulator` インターフェースを実装しており、メインループ（`main.ts`）を汚さずに新しいアルゴリズムを容易に拡張・プラグインできる疎結合な設計（Stateパターン準拠）を採用しています。
- **High Performance**: 衝突判定においては `Math.sqrt` の排除（距離の2乗比較）や空間分割グリッドによるレーダー絞り込みを行い、ブラウザ上での描画負荷を最小限に抑えています。
